# JaneOne
微信工具类小程序【简一】：简单实用，一个就够了，这便是简一诞生的初衷！
### 已实现部分功能：
- 初步的页面设计
- 初步的功能模块分类确认
- 音乐播放功能(调式阶段)
- 快递查询
- 食谱查询
- 药品说明书
- BMI指数
- about页面
